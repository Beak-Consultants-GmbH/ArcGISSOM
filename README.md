# ArcGISSOM
ArcGISSOM is an Esri ArcGIS Toolbox which performs *unsupervised self-organizing maps (SOM)* clustering of 2-dimensional GIS data. It currently runs under ArcGIS 10.6.1. The toolbox has been created in the frame of the project [NEXT - New EXploration Technologies](https://www.new-exploration.tech). This project has received funding from the European Union’s Horizon 2020 research and innovation programme under Grant Agreement No. 776804 — H2020‐SC5‐2017.
